3. Git
Trước khi làm việc với git, cần cấu hình mặc định
- git config --global user.name "<ten_bạn">: config tên người dùng
- git config --global user.email "<tên_email">: config email của bạn
- git config --global init.defaultBranch main: config nhánh mặc định

5. Cách đưa code lên github
    a. Khởi tạo repo local: git init
    b. Tạo repo GitHub và liên kết với repo local: git remote add origin <url>
    c. Thêm file vào staging: git add .
    d. Commit file: git commit -m "<message>"
    e. Push code: git push origin main
